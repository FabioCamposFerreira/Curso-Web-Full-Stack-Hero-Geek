# 📖 MANUAL DE USO - Share Products

## 🎯 Visão Geral

O Share Products permite criar listas de compras/produtos e compartilhá-las via QR Code.

---

## 📱 TELA 1: PRODUTOS

### O que você vê:
- Lista de todos os produtos disponíveis
- Preço de cada produto
- Descrição (quando disponível)
- Filtro de pesquisa
- Contador de itens no carrinho

### O que você pode fazer:

#### 1. Buscar Produtos
```
🔍 Filtrar: [Digite aqui]
```
- Digite parte do nome do produto
- A lista atualiza automaticamente
- Clique em "Limpar" para ver todos os produtos novamente

#### 2. Adicionar ao Carrinho
```
[Produto]
R$ XX.XX
[🛒 Adicionar]  ← Clique aqui
```
- Clique em "🛒 Adicionar" em qualquer produto
- O botão muda para "✓ No carrinho"
- Contador no topo aumenta

#### 3. Ir para o Carrinho
```
[🛒 Carrinho (X)]  ← Clique aqui
```
- Clique no botão verde no topo
- Número mostra quantos itens diferentes você tem

#### 4. Ler QR Code de Outros
```
[📷 Ler QR Code]  ← Clique aqui
```
- Clique no botão roxo no topo
- Para importar lista de outras pessoas

---

## 🛒 TELA 2: CARRINHO

### O que você vê:
- Lista de produtos selecionados
- Quantidade de cada produto
- Preço unitário
- Valor total de cada item
- **Valor total geral** (em destaque)

### O que você pode fazer:

#### 1. Ajustar Quantidade

**Usando botões +/-:**
```
[-] [2] [+]
```
- **[-]** diminui em 1
- **[+]** aumenta em 1
- Se chegar a 0, remove o item

**Editando diretamente:**
```
[-] [2] [+]
     ↑
  Clique aqui
```
- Clique no número
- Digite a quantidade desejada
- Pressione Enter ou clique fora

#### 2. Remover Item
```
[🗑️]  ← Clique aqui
```
- Remove o produto do carrinho
- Pergunta confirmação

#### 3. Limpar Todo o Carrinho
```
[🗑️ Limpar Carrinho]  ← Botão vermelho
```
- Remove TODOS os itens
- Pede confirmação

#### 4. Compartilhar
```
[📤 Compartilhar]  ← Botão roxo
```
- Gera código único
- Mostra QR Code
- Permite compartilhar sua lista

#### 5. Voltar
```
[← Voltar]
```
- Retorna à tela de produtos
- Seu carrinho é mantido

---

## 📤 TELA 3: QR CODE

### O que você vê:
- Código alfanumérico (ex: A1B2C3D4E5F6)
- QR Code visual (imagem)
- Mensagem de sucesso

### O que você pode fazer:

#### 1. Copiar Código
```
A1B2C3D4E5F6
[📋 Copiar Código]  ← Clique aqui
```
- Copia o código para a área de transferência
- Cole em WhatsApp, e-mail, etc.

#### 2. Compartilhar QR Code
- Tire print da tela
- Ou use ferramenta de captura
- Compartilhe a imagem

#### 3. Criar Novo Carrinho
```
[🛒 Novo Carrinho]
```
- Limpa o carrinho atual
- Volta para tela de produtos
- Pronto para nova lista

---

## 📷 TELA 4: LER QR CODE

### Opção 1: Digitar Código

```
Digite o código do carrinho:
[_____________]  [📥 Carregar]
```

**Como usar:**
1. Cole ou digite o código recebido
2. Clique em "📥 Carregar"
3. Carrinho é carregado automaticamente

### Opção 2: Ler de Imagem

```
[📁 Abrir Imagem]
```

**Como usar:**
1. Clique no botão
2. Selecione uma imagem do QR Code
3. Carrinho é carregado automaticamente

**Formatos aceitos:** PNG, JPG, JPEG, BMP

### Opção 3: Usar Câmera

```
[📷 Usar Câmera]
```

**Como usar:**
1. Clique no botão
2. Permita acesso à câmera
3. Aponte para o QR Code
4. Aguarde detecção automática
5. Pressione ESC para cancelar

## 💡 DICAS E TRUQUES

### 🎯 Planejamento de Compras
1. Adicione todos os produtos necessários
2. Ajuste as quantidades
3. Veja o total antes de ir às compras
4. Compartilhe com família/amigos

### 👥 Compras em Grupo
1. Crie sua lista
2. Compartilhe o QR Code
3. Outros podem ver o que você precisa
4. Evita compras duplicadas

### 📊 Controle de Gastos
- O total é calculado automaticamente
- Veja quanto vai gastar antes de comprar
- Ajuste quantidades para caber no orçamento

### 🔄 Listas Recorrentes
1. Crie uma lista padrão (ex: compras do mês)
2. Salve o código
3. Reutilize sempre que precisar
4. Ajuste quantidades conforme necessário

## ⚠️ PROBLEMAS COMUNS

### "Carrinho não encontrado"
**Possíveis causas:**
- Código digitado incorretamente
- QR Code ilegível/corrompido

**Solução:**
- Verifique se copiou o código completo
- Tente novamente com imagem de melhor qualidade

### Câmera não abre
**Possíveis causas:**
- Câmera não conectada
- Permissões negadas
- Outra aplicação usando a câmera

**Solução:**
- Conecte/ative a câmera
- Permita acesso nas configurações
- Feche outros programas que usam câmera

### QR Code não é detectado
**Possíveis causas:**
- Imagem de baixa qualidade
- QR Code muito pequeno/grande
- Iluminação ruim (câmera)

**Solução:**
- Use imagem de melhor resolução
- Aproxime/afaste a câmera
- Melhore a iluminação

## 📞 PERGUNTAS FREQUENTES

### Posso editar uma lista compartilhada?
Não. Ao carregar uma lista compartilhada, você cria uma cópia no seu carrinho. Edições não afetam o original.

### Os códigos expiram?
Não. Uma vez gerado, o código funciona permanentemente.

### Posso ter múltiplos carrinhos salvos?
Sim. Cada vez que compartilha, um novo código é gerado. Você pode ter quantos quiser.

### Quantos itens posso adicionar?
Sem limite. Adicione quantos produtos precisar.

### Os preços são atualizados automaticamente?
Não. Os preços são fixos no momento da criação do carrinho.

Total: R$ 42,90

```
Código: 3F7A2B9E1C4D
QR Code gerado ✓
```

**Passo 3:** Enviar para família
```
WhatsApp: "Essa é a lista do mês: 3F7A2B9E1C4D"
```

**Passo 4:** Família recebe e carrega
```
1. Abre Share Products
2. Clica em "Ler QR Code"
3. Cola o código
4. Lista carregada! ✓
```
